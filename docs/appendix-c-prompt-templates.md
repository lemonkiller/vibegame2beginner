## 通用模板

### 创建新游戏

```
我在用 LÖVE2D 做一个[游戏名称]游戏。请帮我创建完整的代码，要求：
- 屏幕大小 [宽]x[高]
- [描述核心玩法]
- [描述游戏规则]
- 有开始界面、游戏界面、结束界面
- 所有代码写在一个 main.lua 中
```

### 添加功能

```
这是我现在的 LÖVE2D 游戏代码：

--- main.lua ---
（粘贴你的代码）

请在此基础上添加以下功能：
- [描述你要添加的功能]
- 不要改变现有的功能
- 保持代码风格一致
```

### 修复Bug

```
我在运行 LÖVE2D 游戏时遇到问题：

错误信息：
（粘贴完整的错误信息，包括行号）

--- main.lua ---
（粘贴你的代码）

请帮我找出问题并修复，同时解释问题产生的原因。
```

## 贪吃蛇 {#贪吃蛇}

### 基础版

```
帮我创建一个 LÖVE2D 贪吃蛇游戏。要求：
- 用方向键控制蛇的移动方向
- 蛇吃到食物后身体变长，分数加1
- 蛇撞到屏幕边缘或自己的身体就游戏结束
- 有开始界面（显示游戏标题和"按空格开始"）
- 有游戏结束界面（显示最终分数和"按R重新开始"）
- 用网格绘制，蛇是绿色的方块，食物是红色的方块
- 屏幕大小 800x600，网格大小 20x20

```

4. 游戏结束时蛇身逐节变红

## 打砖块 {#打砖块}

### 基础版

```
帮我创建一个 LÖVE2D 打砖块游戏。要求：
- 屏幕大小 800x600
- 挡板在底部，用左右方向键或鼠标控制
- 球碰到墙壁、挡板、砖块会反弹
- 砖块有5行8列，不同行颜色不同（红、橙、黄、绿、蓝）
- 砖块被球碰到后消失，加10分
- 球掉到底部失去一条命，共3条命
- 有开始界面和游戏结束界面
- 球碰到挡板不同位置时反弹角度不同（碰到边缘弹得更偏）
- 砖块碎裂时有碎片粒子效果
- 所有代码写在一个 main.lua 中
```

### 增加砖块种类

```
在打砖块游戏中，请修改砖块系统：
- 第一行(红)的砖块需要打2次才能碎
- 第二行(橙)的砖块需要打2次
- 下面三行打1次就碎
- 多次命中的砖块被打过一次后颜色变暗，作为视觉提示
```

## RPG {#RPG}

### 框架版

```
帮我创建一个 LÖVE2D 的文字冒险RPG框架。要求：
- 屏幕大小 800x600
- 顶部是场景区域，显示当前地点和场景描述
- 底部是对话区域，显示NPC对话和选项
- 对话逐字出现（打字机效果，每秒40字）
- 按空格跳过打字效果
- 对话选项用数字键1/2/3选择
- 支持多个房间和多个NPC
- 暂时不需要图片，全部用文字渲染
```

### 生成NPC对话内容

```
请为我的RPG游戏生成一个NPC的对话数据。NPC是一个[角色描述]，他知道[秘密/信息]。

格式要求（Lua代码）：
```lua
{
    name = "NPC名字",
    greeting = "初次见面的开场白",
    dialogues = {
        [1] = {
            text = "对话文字",
            choices = {
                {text = "选项1", next = 2},
                {text = "选项2", next = 3},
            }
        },
        -- 更多对话...
    }
}
```

要求：
- 至少3轮对话
- 每轮2~3个选择
- 不同选择走向不同的对话
- 对话风格[正式/幽默/神秘/紧张]
```

### 生成战斗系统

```
帮我为 LÖVE2D RPG 添加回合制战斗系统：
- 玩家有HP、攻击力、防御力
- 敌人有HP、攻击力
- 每回合选择：[1]攻击 [2]防御 [3]使用药水
- 攻击：对敌人造成伤害
- 防御：下回合受到的伤害减半
- 药水：回复HP（有限次数）
- 战斗界面显示双方HP条和战斗信息
```

## 平台跳跃游戏 {#平台跳跃游戏}

### 基础版

```
帮我创建一个 LÖVE2D 平台跳跃游戏，风格参考超级马里奥。要求：
- 屏幕大小 800x600
- 用左右方向键移动，空格键跳跃（按住跳跃键可以跳得更高，松开则快速下落，实现可变跳跃高度）
- 角色有重力，会自然下落，站在平台上不会掉下去
- 平台用绿色方块绘制，地面是连续的底部平台
- 角色是红色方块，有简单的奔跑动画（左右交替迈腿）
- 金币散布在关卡中，角色碰到金币就拾取，分数加1，金币有旋转动画
- 敌人是棕色方块，在平台上来回巡逻，碰到边缘自动转向
- 角色从上方踩到敌人可以消灭敌人，敌人被踩时有压扁动画然后消失
- 角色从侧面碰到敌人则失去一条命，共3条命
- 角色掉出屏幕底部也算失去一条命
- 有开始界面（显示游戏标题和"按空格开始"）
- 有游戏结束界面（显示最终分数和"按R重新开始"）
- 所有代码写在一个 main.lua 中
```

### 添加功能 {#平台跳跃-添加功能}

```
在平台跳跃游戏中，请添加以下功能：
1. 角色跳跃时有拉伸变矮的起跳动画和着地时的压扁恢复动画
2. 金币拾取时产生4个向外飞散的金色小粒子
3. 踩敌人时角色自动弹跳起来（反弹跳）
4. 添加移动平台（水平来回移动的平台，角色站在上面会随之移动）
5. 顶部显示金币数量和剩余生命数
6. 角色死亡时先向上弹起再掉出屏幕，而不是直接消失
```

## 射击游戏 {#射击游戏}

### 基础版

```
帮我创建一个 LÖVE2D 射击游戏，风格参考太空侵略者。要求：
- 屏幕大小 800x600
- 用左右方向键移动飞船，空格键发射子弹（按住可以连射，每秒最多4发）
- 玩家飞船在屏幕底部，用蓝色三角形绘制
- 子弹是黄色细长矩形，向上飞行，飞出屏幕后回收
- 敌人从屏幕顶部生成，用红色倒三角形绘制
- 敌人分为3排，每排8个，整体缓慢左右移动，碰到屏幕边缘后所有敌人下移一行再反向移动
- 敌人偶尔会向下发射子弹（每3秒随机选一个敌人发射），敌人子弹是红色小圆
- 玩家被敌人子弹击中失去一条命，共3条命
- 子弹击中敌人则敌人爆炸消失，加10分
- 所有敌人被消灭后进入下一关：敌人移动速度加快，敌人射击频率提高
- 有开始界面（显示游戏标题和"按空格开始"）
- 有游戏结束界面（显示最终分数、到达关卡和"按R重新开始"）
- 所有代码写在一个 main.lua 中
```

### 添加功能 {#射击游戏-添加功能}

```
在射击游戏中，请添加以下功能：
1. 敌人被击中时有爆炸粒子效果（6个橙色粒子向外飞散并逐渐消失）
2. 每隔30秒在屏幕顶部随机位置出现一个蓝色菱形道具，向下缓慢飘落
3. 道具类型：S=加速射击（5秒内射击速度翻倍）、+=额外生命
4. 左上角显示当前分数、关卡、剩余生命
5. 玩家飞船被击中时有0.5秒的无敌闪烁时间
6. 新关卡开始时屏幕中央显示"WAVE X"文字2秒，期间游戏暂停
```

代码：
