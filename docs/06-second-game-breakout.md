# 第6章 第二个游戏：打砖块

> 学会了"移动-碰撞"，现在学"反弹-消除"。这两个概念能覆盖80%的2D游戏。

## 这个游戏做什么？

打砖块（Breakout）——经典中的经典：

- 一个挡板在屏幕底部，用左右方向键（或鼠标）控制
- 一个球在屏幕上弹来弹去
- 球碰到砖块，砖块消失，得分
- 球掉到屏幕底部，失去一条命
- 清除所有砖块则通关

> 📍 打砖块在复杂度地图的**偏右中间**——系统中等（碰撞+反弹+生命+粒子），但内容少。

## 新概念：反弹和碰撞区

贪吃蛇教了你"格子坐标"的碰撞——两个东西在同一个格子就算碰上了。打砖块需要更精细的碰撞检测，因为球是像素级移动的。

### AABB碰撞检测

游戏里最常用的碰撞检测叫 AABB（Axis-Aligned Bounding Box），翻译成人话就是"矩形碰矩形"：

```
如果 A 的右边 > B 的左边，且
   A 的左边 < B 的右边，且
   A 的下边 > B 的上边，且
   A 的上边 < B 的下边
那么 A 和 B 碰撞了
```

听起来复杂，用代码写就一个函数：

```lua
-- 检测两个矩形是否碰撞
function check_collision(a, b)
    return a.x < b.x + b.w and
           a.x + a.w > b.x and
           a.y < b.y + b.h and
           a.y + a.h > b.y
end
```

这行代码你不需要背，AI会帮你写。但你得知道"碰撞检测"这个概念，这样你才能告诉AI你要什么。

### 反弹

球碰到东西后要反弹。简单规则：
- 碰到左右墙——水平方向反转
- 碰到上下墙或挡板/砖块——垂直方向反转

```lua
-- 碰到左右墙，水平速度反转
ball.vx = -ball.vx

-- 碰到挡板/砖块，垂直速度反转
ball.vy = -ball.vy
```

实际上，为了让游戏更有趣，球碰到挡板的不同位置时，反弹角度应该不同——碰到挡板边缘弹得更偏，碰到中间弹得更直。这个细节可以让AI帮你加。

## 创建你的打砖块

在 Cursor 中输入：

> 帮我创建一个 LÖVE2D 打砖块游戏。要求：
> - 屏幕大小 800x600
> - 挡板在底部，用左右方向键或鼠标控制
> - 球碰到墙壁、挡板、砖块会反弹
> - 砖块有5行8列，不同行颜色不同
> - 砖块被球碰到后消失，加10分
> - 球掉到底部失去一条命，共3条命
> - 有开始界面和游戏结束界面
> - 球碰到挡板不同位置时反弹角度不同
> - 所有代码写在 main.lua 中

运行 `love .` 看效果，然后逐步调整。

## 循序渐进

和贪吃蛇一样，你也可以分步做：

### 第一步：画出挡板和球

> 帮我创建一个 LÖVE2D 项目，屏幕上有一个可以用方向键左右移动的挡板，和一个静止的球

### 第二步：让球动起来

> 让球自动移动，碰到左右和上方墙壁时反弹，掉到底部时停止

### 第三步：添加砖块

> 在屏幕上方添加5行8列的砖块，球碰到砖块时砖块消失

### 第四步：完善游戏流程

> 添加开始界面（按空格开始）、生命系统（3条命）、分数、游戏结束界面

## 微调你的游戏

### 让挡板更顺滑

> 把挡板的移动方式从方向键步进改成平滑移动，也支持鼠标控制挡板位置

### 增加砖块强度

> 把砖块改成两种：蓝色砖块打一下就碎，银色砖块需要打两下。不同行用不同颜色。

### 添加粒子效果

> 砖块被击碎时产生碎片粒子效果，粒子向四周飞散然后消失

### 添加道具

> 某些砖块打碎后掉落道具：
> - 红色道具：挡板变长
> - 蓝色道具：球变慢
> - 绿色道具：多一个球

## 变体挑战

打砖块的核心技术是"AABB碰撞 + 反弹物理"。这两个东西能做的游戏比你想的多得多：

| 挑战 | 变化点 | 难度 |
|------|--------|------|
| 🧱 多关卡打砖块 | 不同关卡砖块排成不同形状 | ⭐ |
| ⚡ 弹幕打砖块 | 砖块会发射子弹下来 | ⭐⭐ |
| 🏓 双挡板打砖块 | 屏幕上下各一个挡板 | ⭐⭐ |
| 🌊 打砖块RPG | 砖块有血量，打砖块赚经验升级挡板 | ⭐⭐⭐ |
| 🧊 物理打砖块 | 加入重力，球会下坠，更真实 | ⭐⭐⭐ |

**起步提示**：

- 多关卡打砖块 → "添加关卡数据表，清完一组砖块加载下一组"
- 弹幕打砖块 → "某些砖块每隔几秒向下射出子弹，挡板要躲避"
- 双挡板打砖块 → "屏幕顶部加第二个挡板，用A/D键控制"
- 打砖块RPG → "挡板有等级和经验值，打砖块加分，升级挡板变长"
- 物理打砖块 → "球加入重力加速度，每帧vy增加一个小值"

> 💡 弹幕打砖块其实就是打砖块+射击游戏的混血。你看，学会碰撞+反弹，能覆盖的游戏种类远超你想象。

## 遇到问题？

| 问题 | 可能的原因 | 解决方法 |
|------|-----------|---------|
| 球穿过砖块不反弹 | 球速太快，一帧跳过了砖块 | 降低球速或用子步进碰撞检测 |
| 球卡在墙里 | 反弹逻辑重复触发 | 加一个碰撞冷却标记 |
| 挡板控制不顺 | 每帧固定步进不跟手 | 改成鼠标控制或持续按键检测 |
| 球一直在水平弹 | 碰到挡板后只反转vy不调整角度 | 根据碰到挡板的位置调整vx |
