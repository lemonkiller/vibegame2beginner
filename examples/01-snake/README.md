# 贪吃蛇

经典贪吃蛇游戏 —— 用方向键控制蛇吃食物，越长越难。

## 快速运行

### 前提条件

已安装 LÖVE2D（版本 11.5 或更高）。如果还没安装，参见[安装指引](../../docs/02-tools.md#第一步安装-love2d)。

### 一键运行

在终端中进入本目录，然后执行：

```bash
love .
```

或者双击本目录，用 LÖVE 打开。

### Windows 用户

如果你把 `.love` 文件关联了 LÖVE，也可以直接把整个文件夹拖到 `love.exe` 上运行。

### macOS 用户

如果是 Homebrew 安装的 LÖVE，在终端中执行：

```bash
love .
```

如果是手动下载的 `.app`，需要把文件夹拖到 LÖVE 应用图标上。

### Linux 用户

```bash
love .
```

如果提示找不到命令，请确认 LÖVE 已安装（如 `sudo apt install love`）。

## 操作说明

| 按键 | 功能 |
|------|------|
| 方向键 ↑↓←→ | 控制蛇的移动方向 |
| 空格键 | 在开始画面按此键进入游戏 |

## 游戏规则

- 蛇吃到红色食物后身体变长，分数 +1
- 每吃一个食物，移动速度略微加快
- 蛇撞到墙壁或自己的身体，游戏结束
- 按空格键回到开始画面，按 R 重新开始

## 代码结构

```
01-snake/
├── main.lua    # 游戏全部逻辑（开始/游戏/结束三个状态）
└── conf.lua    # 窗口配置（800x600，标题"贪吃蛇"）
```

`main.lua` 采用状态机模式：`state` 变量控制游戏在 `start`、`playing`、`gameover` 三个状态间切换。新手可以重点看 `love.load()`、`love.update(dt)`、`love.draw()` 三个核心函数。
