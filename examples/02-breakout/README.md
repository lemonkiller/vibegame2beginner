# 打砖块

经典打砖块游戏 —— 弹球消除砖块，别让球掉出去。

## 快速运行

### 前提条件

已安装 LÖVE2D（版本 11.5 或更高）。如果还没安装，参见[安装指引](../../docs/02-tools.md#第一步安装-love2d)。

### 一键运行

在终端中进入本目录，然后执行：

```bash
love .
```

或者双击本目录，用 LÖVE 打开。

### Windows 用户

如果你把 `.love` 文件关联了 LÖVE，也可以直接把整个文件夹拖到 `love.exe` 上运行。

### macOS 用户

如果是 Homebrew 安装的 LÖVE，在终端中执行：

```bash
love .
```

如果是手动下载的 `.app`，需要把文件夹拖到 LÖVE 应用图标上。

### Linux 用户

```bash
love .
```

如果提示找不到命令，请确认 LÖVE 已安装（如 `sudo apt install love`）。

## 操作说明

| 按键 | 功能 |
|------|------|
| ←→ 方向键 或 A/D 键 | 控制挡板左右移动 |
| 鼠标 | 按住鼠标左键，挡板跟随鼠标 |
| 空格键 | 发射球 / 开始游戏 |

## 游戏规则

- 用挡板接住弹跳的球，反弹回去消除砖块
- 不同颜色行的砖块只是视觉区分，分数相同（每个 10 分）
- 共 3 条命，球掉出底部扣 1 条命
- 消除全部砖块即通关
- 砖块碎裂时有粒子爆散效果

## 代码结构

```
02-breakout/
├── main.lua    # 游戏全部逻辑（含碰撞检测、粒子系统）
└── conf.lua    # 窗口配置（800x600，标题"打砖块"）
```

`main.lua` 包含了圆形-矩形碰撞检测（`check_ball_rect_collision`）和粒子效果系统，是学习碰撞和视觉效果的好素材。
