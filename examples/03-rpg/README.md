# RPG 对话冒险

AI驱动的RPG对话游戏 —— 用选择推进剧情，体验AI讲故事。

## 快速运行

### 前提条件

已安装 LÖVE2D（版本 11.5 或更高）。如果还没安装，参见[安装指引](../../docs/02-tools.md#第一步安装-love2d)。

### 一键运行

在终端中进入本目录，然后执行：

```bash
love .
```

或者双击本目录，用 LÖVE 打开。

### Windows 用户

如果你把 `.love` 文件关联了 LÖVE，也可以直接把整个文件夹拖到 `love.exe` 上运行。

### macOS 用户

如果是 Homebrew 安装的 LÖVE2D，在终端中执行：

```bash
love .
```

如果是手动下载的 `.app`，需要把文件夹拖到 LÖVE 应用图标上。

### Linux 用户

```bash
love .
```

如果提示找不到命令，请确认 LÖVE 已安装（如 `sudo apt install love`）。

## 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标点击 | 选择对话选项 |
| 点击画面 | 推进对话文字 |

## 游戏规则

- 阅读NPC的对话文字
- 在选项出现时点击你想要的回复
- 不同选择会导向不同的剧情走向
- 这是一个对话驱动的冒险游戏，重点在故事体验

## 代码结构

```
03-rpg/
├── main.lua    # 游戏全部逻辑（对话系统、选择分支）
└── conf.lua    # 窗口配置
```

`main.lua` 实现了一个基于对话树的系统：每个对话节点包含文本和可选的分支选项，选择后跳转到下一节点。新手可以重点看对话数据结构和 `advance_dialogue()` 函数。
