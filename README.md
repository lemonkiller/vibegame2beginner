# 用AI做游戏

> 不会写代码没关系，学会"说清楚你想要什么"就够了。

## 这本手册适合谁？

- 不会编程但想自己做游戏的普通人
- 有创意想法但被技术门槛劝退的游戏爱好者
- 想尝试AI编程但不知道从哪开始的新手
- 设计师、策划、产品经理等非技术岗位的人

## 你不需要什么？

- 不需要任何编程基础
- 不需要安装复杂的开发环境
- 不需要背诵代码语法
- 不需要花几千块买课程

## 你需要什么？

- 一台电脑（Windows / Mac / Linux 都行）
- 会打字、会描述你想要的东西
- 愿意动手尝试、不怕出错

## 学习路径

```
第0章  前置认知        → 搞清楚AI能做什么、不能做什么
第1章  选你的武器      → 找到适合你的工具
第2章  学会和AI说话    → 掌握提需求的"语言"
第3章  第一个游戏      → 2小时做出可玩的贪吃蛇
第4章  第二个游戏      → 理解游戏逻辑，做打砖块
第5章  第三个游戏      → 让AI成为游戏的一部分（RPG对话冒险）
第6章  让游戏变好看    → AI生成美术素材
第7章  让游戏变好听    → AI生成音乐和音效
第8章  从原型到发布    → 让别人也能玩到你的游戏
第9章  避坑指南        → 少走弯路的经验总结
```

## 技术栈

本手册使用 **LÖVE2D**（一个用Lua写的2D游戏框架）作为主线技术栈，原因如下：

- **Lua 是最简单的编程语言之一**，语法接近自然语言
- **LÖVE2D 专为2D游戏设计**，不需要理解复杂的引擎概念
- **AI 对 Lua 的理解非常好**，生成的代码质量高
- **跨平台**，一次编写，Windows/Mac/Linux/手机都能跑
- **社区活跃**，遇到问题容易找到答案
- **性能足够**，2D同屏精灵上万无压力（《Balatro》即用LÖVE2D开发，销量百万级）

## 代码示例

每个章节的完整代码在 `examples/` 目录下：

```
examples/
├── 01-snake/       # 第3章贪吃蛇完整代码
├── 02-breakout/    # 第4章打砖块完整代码
└── 03-rpg/         # 第5章RPG对话冒险完整代码
```

### 如何运行示例

**前提**：已安装 [LÖVE2D](https://love2d.org)（11.5 或更高版本），安装方法见[第1章](docs/01-tools.md)。

在终端中进入对应目录，执行：

```bash
# 运行贪吃蛇
cd examples/01-snake && love .

# 运行打砖块
cd examples/02-breakout && love .

# 运行RPG对话冒险
cd examples/03-rpg && love .
```

> **Windows 提示**：如果命令行找不到 `love`，把整个游戏文件夹拖到 `love.exe` 上也能运行。

每个例子目录下都有独立的 `README.md`，包含操作说明和代码结构介绍。
